# ML-CreditWise-Loan-System

## 📌 Overview

**ML-CreditWise-Loan-System** is an end-to-end supervised machine learning project that predicts loan approval status based on historical applicant data. The project covers the complete ML lifecycle, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation using multiple classification algorithms.

## 🚀 Features

* Binary classification for **loan approval prediction**
* In-depth **Exploratory Data Analysis (EDA)**
* **Data preprocessing & feature engineering**
* Implementation and comparison of multiple ML models:

  * K-Nearest Neighbors (KNN)
  * Logistic Regression
  * Naive Bayes
* Performance evaluation using industry-standard metrics

## 📊 Dataset Description

The dataset contains historical loan application records with both demographic and financial attributes of applicants.

### 🔹 Key Features

* **ApplicantIncome** – Income of the applicant
* **CoapplicantIncome** – Income of the co-applicant
* **LoanAmount** – Loan amount requested
* **Loan_Amount_Term** – Loan repayment period
* **Credit_History** – Credit history status (0/1)
* **Gender** – Applicant gender
* **Married** – Marital status
* **Dependents** – Number of dependents
* **Education** – Education level
* **Self_Employed** – Employment type
* **Property_Area** – Property location (Urban/Rural/Semiurban)

### 🎯 Target Variable

* **Loan_Status** – Indicates whether the loan was **approved (1)** or **rejected (0)**

## 🧠 Machine Learning Models Used

* **Logistic Regression**
* **K-Nearest Neighbors (KNN)**
* **Naive Bayes**

## 📈 Model Evaluation Metrics

Model performance is evaluated using:

* **Precision**
* **Recall**
* **F1-Score**

These metrics provide a balanced evaluation, especially for imbalanced datasets.
